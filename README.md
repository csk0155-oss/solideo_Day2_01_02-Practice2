# 여행 개인화 앱 🌍

맞춤형 여행 계획을 위한 스마트 여행 도우미 애플리케이션입니다.

## 주요 기능 ✨

### 1. 여행 정보 입력
- 출발지와 도착지 입력
- 출발 시간 선택
- 여행 기간 설정

### 2. 실시간 최적 경로 안내
- 실시간 시각 업데이트 (1초마다 갱신)
- 출발 시간에 따른 교통 상황 예측
- 예상 소요 시간 계산
- 추천 경로 안내
- 교통 혼잡도 정보 제공

### 3. 맛집 추천 (평점순)
- 도착지별 맞춤 맛집 정보
- 평점 높은 순으로 자동 정렬
- 상세 정보 제공 (카테고리, 주소, 대표 메뉴, 가격대)
- 시각적 별점 표시

### 4. 취향 기반 여행지 추천
- 6가지 취향 카테고리
  - 🌳 자연/힐링
  - 🏛️ 문화/역사
  - 🏃 액티비티
  - 🛍️ 쇼핑
  - 📸 포토스팟
  - 🏘️ 로컬 체험
- 취향 선택 시 맞춤형 여행지 추천
- 태그 및 상세 설명 제공

## 지원 도시 🗺️

현재 다음 도시의 맛집 및 여행지 정보를 제공합니다:
- **서울**: 경복궁, 남산, 명동, 광장시장 등
- **부산**: 해운대, 감천문화마을, 자갈치시장 등
- **제주**: 한라산, 성산일출봉, 우도 등
- **강릉**: 경포해변, 정동진, 안목 커피거리 등

## 사용 방법 📱

1. **웹 브라우저에서 index.html 열기**
   ```bash
   # 프로젝트 디렉토리에서
   # 브라우저로 index.html 파일을 직접 열거나
   # 로컬 서버 실행 (선택사항)
   python -m http.server 8000
   # 또는
   python3 -m http.server 8000
   ```

2. **여행 정보 입력**
   - 출발지 입력 (예: 서울)
   - 도착지 입력 (예: 부산)
   - 출발 시간 선택
   - 여행 기간 입력 (일 단위)
   - "여행 계획 시작하기" 버튼 클릭

3. **실시간 경로 확인**
   - 현재 시각이 실시간으로 업데이트됩니다
   - 교통 상황에 따른 예상 소요 시간 확인
   - 추천 경로 확인

4. **맛집 정보 확인**
   - 평점 높은 순으로 정렬된 맛집 리스트 확인
   - 각 맛집의 상세 정보 확인

5. **취향 선택 및 여행지 추천**
   - 원하는 취향 버튼 클릭
   - 맞춤형 여행지 리스트 확인
   - 각 여행지의 태그와 설명 확인

## 주요 특징 🎯

### 실시간 업데이트
- 1초마다 현재 시각 갱신
- 30초마다 교통 상황 재계산
- 시간대별 교통 혼잡도 시뮬레이션

### 스마트 교통 예측
- 출퇴근 시간대 (7-9시, 18-20시): +30% 소요 시간
- 점심 시간대 (12-13시): +15% 소요 시간
- 심야 시간대 (22-06시): -15% 소요 시간

### 사용자 친화적 UI
- 직관적인 인터페이스
- 부드러운 애니메이션 효과
- 카드형 레이아웃
- 호버 효과
- 자동 스크롤

### 평점 기반 추천
- 맛집은 자동으로 평점 높은 순 정렬
- 시각적 별점 표시 (⭐✨)
- 상세한 정보 제공
