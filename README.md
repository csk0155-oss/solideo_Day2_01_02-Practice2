# 여행 개인화 앱 🌍

맞춤형 여행 계획을 위한 스마트 여행 도우미 애플리케이션입니다.

## 주요 기능 ✨

### 1. 여행 정보 입력
- 출발지와 도착지 입력
- 출발 시간 선택
- 여행 기간 설정

### 2. 실시간 최적 경로 안내
- 실시간 시각 업데이트 (1초마다 갱신)
- 출발 시간에 따른 교통 상황 예측
- 예상 소요 시간 계산
- 추천 경로 안내
- 교통 혼잡도 정보 제공

### 3. 맛집 추천 (평점순)
- 도착지별 맞춤 맛집 정보
- 평점 높은 순으로 자동 정렬
- 상세 정보 제공 (카테고리, 주소, 대표 메뉴, 가격대)
- 시각적 별점 표시

### 4. 취향 기반 여행지 추천
- 6가지 취향 카테고리
  - 🌳 자연/힐링
  - 🏛️ 문화/역사
  - 🏃 액티비티
  - 🛍️ 쇼핑
  - 📸 포토스팟
  - 🏘️ 로컬 체험
- 취향 선택 시 맞춤형 여행지 추천
- 태그 및 상세 설명 제공

## 지원 도시 🗺️

현재 다음 도시의 맛집 및 여행지 정보를 제공합니다:
- **서울**: 경복궁, 남산, 명동, 광장시장 등
- **부산**: 해운대, 감천문화마을, 자갈치시장 등
- **제주**: 한라산, 성산일출봉, 우도 등
- **강릉**: 경포해변, 정동진, 안목 커피거리 등

## 사용 방법 📱

### 웹 브라우저에서 사용

1. **로컬 서버 실행**
   ```bash
   # 프로젝트 디렉토리에서
   python3 -m http.server 8000
   # 또는
   python -m http.server 8000
   ```

2. **브라우저에서 접속**
   - http://localhost:8000 접속

### 앱으로 설치하기 📲

#### Android (Chrome)
1. 웹사이트 접속
2. 우측 상단 "앱으로 설치하기" 버튼 클릭
3. 또는 주소창 우측의 "+" 아이콘 클릭
4. "설치" 버튼 클릭
5. 홈 화면에 앱 아이콘 생성됨

#### iOS (Safari)
1. Safari에서 웹사이트 접속
2. 하단 공유 버튼(⬆️) 클릭
3. "홈 화면에 추가" 선택
4. "추가" 클릭
5. 홈 화면에 앱 아이콘 생성됨

#### Desktop (Chrome, Edge)
1. 웹사이트 접속
2. 주소창 우측의 설치 아이콘(💻+) 클릭
3. "설치" 버튼 클릭
4. 독립 실행형 앱으로 실행됨

### 아이콘 생성하기

**자동 생성 (권장)**
1. 브라우저에서 `generate-icons.html` 파일 열기
2. "모든 아이콘 생성" 버튼 클릭
3. 각 아이콘을 `icons/` 폴더에 저장
4. 파일명: `icon-72x72.png`, `icon-96x96.png`, 등

### 앱 기능 사용

1. **여행 정보 입력**
   - 출발지 입력 (예: 서울)
   - 도착지 입력 (예: 부산)
   - 출발 시간 선택
   - 여행 기간 입력 (일 단위)
   - "여행 계획 시작하기" 버튼 클릭

2. **실시간 경로 확인**
   - 현재 시각이 실시간으로 업데이트됩니다
   - 교통 상황에 따른 예상 소요 시간 확인
   - 추천 경로 확인

3. **맛집 정보 확인**
   - 평점 높은 순으로 정렬된 맛집 리스트 확인
   - 각 맛집의 상세 정보 확인

4. **취향 선택 및 여행지 추천**
   - 원하는 취향 버튼 클릭
   - 맞춤형 여행지 리스트 확인
   - 각 여행지의 태그와 설명 확인

## 프로젝트 구조 📁

```
solideo_Day2_01_02-Practice2/
├── index.html              # 메인 HTML 파일
├── offline.html            # 오프라인 페이지
├── generate-icons.html     # 아이콘 생성 도구
├── manifest.json           # PWA 매니페스트
├── service-worker.js       # Service Worker (오프라인 지원)
├── css/
│   └── style.css          # 스타일시트
├── js/
│   ├── app.js            # 메인 애플리케이션 로직
│   ├── data.js           # 맛집 및 여행지 데이터
│   ├── route.js          # 경로 최적화 기능
│   ├── restaurant.js     # 맛집 추천 기능
│   └── attractions.js    # 여행지 추천 기능
├── icons/
│   ├── icon.svg          # SVG 아이콘 원본
│   └── icon-*x*.png      # 다양한 크기의 PNG 아이콘
├── LICENSE.md
└── README.md
```

## 기술 스택 🛠️

- **HTML5**: 웹 구조
- **CSS3**: 스타일링 및 애니메이션
- **JavaScript (ES6+)**: 애플리케이션 로직
- **PWA (Progressive Web App)**: 앱 설치 및 오프라인 지원
  - Service Worker: 오프라인 캐싱
  - Web App Manifest: 앱 메타데이터
  - Cache API: 리소스 캐싱
- **반응형 디자인**: 모바일 및 데스크톱 지원

## 주요 특징 🎯

### PWA 기능
- **앱 설치**: 홈 화면에 추가하여 네이티브 앱처럼 사용
- **오프라인 지원**: 인터넷 연결 없이도 기본 기능 사용 가능
- **자동 업데이트**: 새 버전 자동 감지 및 업데이트
- **푸시 알림**: 여행 정보 알림 (선택사항)
- **빠른 로딩**: 캐시를 활용한 빠른 페이지 로딩
- **독립 실행**: 브라우저 UI 없이 독립 실행

### 실시간 업데이트
- 1초마다 현재 시각 갱신
- 30초마다 교통 상황 재계산
- 시간대별 교통 혼잡도 시뮬레이션

### 스마트 교통 예측
- 출퇴근 시간대 (7-9시, 18-20시): +30% 소요 시간
- 점심 시간대 (12-13시): +15% 소요 시간
- 심야 시간대 (22-06시): -15% 소요 시간

### 사용자 친화적 UI
- 직관적인 인터페이스
- 부드러운 애니메이션 효과
- 카드형 레이아웃
- 호버 효과
- 자동 스크롤

### 평점 기반 추천
- 맛집은 자동으로 평점 높은 순 정렬
- 시각적 별점 표시 (⭐✨)
- 상세한 정보 제공
